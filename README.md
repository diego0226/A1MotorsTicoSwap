# 🚀 Instalación

```bash
# Clonar repositorio
git clone https://github.com/TU-USUARIO/a1motorsticoswap.git

# Entrar al proyecto
cd a1motorsticoswap

# Instalar dependencias
npm install

# Ejecutar entorno de desarrollo
npm run dev
```
